# 🚀 AWS Three Tier Architecture Project

This project demonstrates deployment of a scalable three-tier web application on AWS.

## 🧱 Architecture
- Application Load Balancer
- Web Tier (Nginx)
- App Tier (Node.js)
- Target Groups & Health Checks
- Public & Private Subnets

## ⚙️ Tech Stack
- AWS EC2
- ALB
- Node.js
- Nginx
- VPC Networking

## 🎯 Features
✔ High availability using multiple AZ  
✔ Load balancing with ALB  
✔ Secure tier separation  
✔ Health checks configured
